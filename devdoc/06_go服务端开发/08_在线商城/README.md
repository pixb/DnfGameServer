## 系统简介
在线商城系统是游戏内官方商城功能，支持玩家使用游戏货币或真实货币购买游戏道具、礼包等。系统提供商品展示、分类浏览、搜索、购买等核心功能，为玩家提供便捷的游戏内购物体验。

## 核心功能
- **商品管理**：支持多种商品类型、价格设置、库存管理
- **分类系统**：多级分类管理，方便玩家浏览
- **搜索功能**：按名称、类型、价格范围搜索商品
- **购买系统**：支持多种货币类型，购买限制，购买记录
- **推荐系统**：智能推荐热门商品、新品等
- **促销活动**：支持限时折扣、促销礼包等活动

## 技术栈
- **后端**：Go 1.20+, gRPC, GORM
- **数据库**：MySQL 5.7+
- **前端**：游戏客户端
