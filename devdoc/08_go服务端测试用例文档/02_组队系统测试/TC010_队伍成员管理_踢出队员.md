# TC010_队伍成员管理_踢出队员

## 测试概述

测试队长踢出队员的功能，确保踢出队员接口能够正常工作。

## 前置条件

1. 已启动Go服务端（端口8082）
2. 已创建测试账号和角色
3. 已获取有效的authKey
4. 玩家是队伍队长
5. 队伍中有其他队员（获取队员的charGuid）

## 测试步骤

1. 发送HTTP POST请求到 `/api/v1/party/kick` 接口，携带authKey和队员charGuid参数
2. 检查返回结果

## 预期结果

1. 接口返回成功
2. error字段为0
3. 队员成功被踢出队伍
