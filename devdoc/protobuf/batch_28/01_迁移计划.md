## 三、数据结构设计

### 3.1 核心消息结构

#### 3.1.1 导师信息

```protobuf
// 导师信息
message PtMenteeInfo {
  uint64 guid = 1;       // 导师GUID
  string name = 2;        // 导师名称
  int32 level = 3;        // 导师等级
  int32 job = 4;          // 导师职业
  int32 relationship = 5;  // 关系
}
```

#### 3.1.2 导师任务信息

```protobuf
// 导师任务信息
message PtMentorQuestInfo {
  int32 quest_id = 1;     // 任务ID
  int32 status = 2;       // 状态
  int32 progress = 3;     // 进度
  int32 total = 4;         // 总数
}
```

#### 3.1.3 拍卖物品价格信息

```protobuf
// 拍卖物品价格信息
message PtAuctionItemPriceInfo {
  int32 item_id = 1;      // 物品ID
  int64 price = 2;         // 价格
  int32 count = 3;         // 数量
  uint64 seller_guid = 4; // 卖家GUID
}
```

#### 3.1.4 角色槽位信息

```protobuf
// 角色槽位信息
message PtCharacterSlotInfo {
  int32 slot_id = 1;      // 槽位ID
  int32 status = 2;        // 状态
  uint64 character_guid = 3; // 角色GUID
  string character_name = 4;   // 角色名称
}
```
