### 3.1 核心消息结构

#### 3.1.1 可堆叠物品信息

```protobuf
// 可堆叠物品信息
message Stackable {
  int32 item_id = 1;      // 物品ID
  int32 count = 2;         // 数量
  int32 max_stack = 3;     // 最大堆叠数
  int64 expire_time = 4;   // 过期时间
}
```

#### 3.1.2 可重新购买的可堆叠物品信息

```protobuf
// 可重新购买的可堆叠物品信息
message RepurchaseStackable {
  int32 item_id = 1;      // 物品ID
  int32 count = 2;         // 数量

  int64 repurchase_time = 4; // 重新购买时间
}
```
