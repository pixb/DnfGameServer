## 技术实现

### 消息类型结构

所有消息类型将按照以下结构定义：

```protobuf
message MessageName {
  int32 error = 1;
  int32 index = 2;
}
```

### 字段说明

- **error**: 错误码字段，类型为int32
- **index**: 索引字段，类型为int32

### 代码生成

使用buf工具生成代码：

```bash
buf generate
```

生成路径：
- Java: `gen/java`
- Go: `../dnf-go-client/gen`
