### 3.1 核心消息结构

#### 3.1.1 物品GUID

```protobuf
// 物品GUID
message PtItemGuid {
  uint64 guid = 1;       // 物品GUID
  int32 item_id = 2;      // 物品ID
  int64 instance_id = 3;   // 实例ID
}
```

#### 3.1.2 槽位附魔经验信息

```protobuf
// 槽位附魔经验信息
message PtSlotEnchantExp {
  int32 slot_id = 1;     // 槽位ID
  int32 exp = 2;          // 经验值
  int32 max_exp = 3;       // 最大经验值
}
```

#### 3.1.3 槽位附魔信息

```protobuf
// 槽位附魔信息
message PtSlotEnchantInfo {
  int32 slot_id = 1;     // 槽位ID
  int32 enchant_level = 2; // 附魔等级
  repeated int32 attributes = 3; // 属性列表
}
```

### 3.2 数据类型映射

| JProtobuf类型 | 标准Protobuf类型 | 备注 |
| :--- | :--- | :--- |
| int | int32 | 整数类型 |
| long | uint64 | 无符号长整数类型 |
| String | string | 字符串类型 |
| boolean | bool | 布尔类型 |
| List<T> | repeated T | 列表类型 |
| 自定义对象 | message | 嵌套消息 |
