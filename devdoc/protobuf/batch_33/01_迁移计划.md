### 3.1 核心消息结构

#### 3.1.1 系统命令

```protobuf
// 系统命令
message SyscmdCommand {
  int32 runtype = 1;     // 运行类型
  int32 targettype = 2;   // 目标类型
  int32 type = 3;         // 类型
  uint64 time = 4;         // 时间
}
```

#### 3.1.2 公会buff升级成本

```protobuf
// 公会buff升级成本
message GuildBuffUpgradeCost {
  int32 itemindex = 1;  // 物品索引
  int32 count = 2;        // 数量
}
```

#### 3.1.3 公会捐赠累积奖励

```protobuf
// 公会捐赠累积奖励
message GuildDonationAccumulateReward {
  int32 clearcount = 1;  // 清除数量

}
```

| JProtobuf类型 | 标准Protobuf类型 | 备注 |
