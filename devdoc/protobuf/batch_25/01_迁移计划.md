## 三、数据结构设计

### 3.1 核心消息结构

#### 3.1.1 角色信息

```protobuf
// 角色信息
message Actor {
  uint64 guid = 1;       // 角色GUID
  string name = 2;        // 角色名称
  int32 level = 3;        // 角色等级
  int32 job = 4;          // 职业
  float x = 5;            // X坐标
  float y = 6;            // Y坐标
  float z = 7;            // Z坐标
  int32 map_id = 8;       // 地图ID
}
```

#### 3.1.2 聊天基础信息

```protobuf
// 聊天基础信息
message ChatBase {
  int32 type = 1;         // 聊天类型
  string content = 2;      // 聊天内容
  uint64 sender_guid = 3;  // 发送者GUID
  string sender_name = 4;   // 发送者名称
  int64 timestamp = 5;      // 时间戳
}
```

#### 3.1.3 子系统信息

```protobuf
// 子系统信息
message Subsystem {
  int32 subsystem_id = 1;  // 子系统ID
  string name = 2;         // 子系统名称
  int32 status = 3;        // 状态
  int32 priority = 4;      // 优先级
}
```

### 3.2 数据类型映射

| JProtobuf类型 | 标准Protobuf类型 | 备注 |
| :--- | :--- | :--- |
| int | int32 | 整数类型 |
| long | uint64 | 无符号长整数类型 |
| String | string | 字符串类型 |
| boolean | bool | 布尔类型 |
| float | float | 浮点数类型 |
| List<T> | repeated T | 列表类型 |
| 自定义对象 | message | 嵌套消息 |
