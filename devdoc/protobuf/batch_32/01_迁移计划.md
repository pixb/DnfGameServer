### 3.1 核心消息结构

#### 3.1.1 自定义数据

```protobuf
// 自定义数据
message CustomData {
  int32 data_type = 1;   // 数据类型

  string key = 3;          // 键
}
```

#### 3.1.2 怪物经验信息

```protobuf
// 怪物经验信息
message MonsterExp {
  int32 monster_id = 1;  // 怪物ID
  int32 exp = 2;          // 经验值
  int32 level = 3;        // 等级
}
```

#### 3.1.3 隐藏聊天信息

```protobuf
// 隐藏聊天信息
message HiddenChattingInfo {

  string name = 2;        // 名称
  int32 type = 3;         // 类型
}
```
