## 三、数据结构设计

### 3.1 核心消息结构

#### 3.1.1 公会符号

```protobuf
// 公会符号
message GuildSymbol {
  int32 symbol_id = 1;    // 符号ID
  string icon = 2;        // 图标
  string name = 3;         // 名称
}
```

#### 3.1.2 avatar物品

```protobuf
// avatar物品
message SdAvatarItem {
  int32 item_id = 1;      // 物品ID
  string name = 2;         // 物品名称
  int32 quality = 3;       // 品质
  repeated int32 attributes = 4; // 属性列表
}
```

#### 3.1.3 排名信息

```protobuf
// 排名信息
message Ranking {
  int32 rank = 1;         // 排名
  uint64 guid = 2;        // 角色GUID
  string name = 3;         // 角色名称
  int32 score = 4;        // 分数
  int32 level = 5;        // 等级
}
```

### 3.2 数据类型映射

| JProtobuf类型 | 标准Protobuf类型 | 备注 |
| :--- | :--- | :--- |
| int | int32 | 整数类型 |
| long | uint64 | 无符号长整数类型 |
| String | string | 字符串类型 |
| boolean | bool | 布尔类型 |
| List<T> | repeated T | 列表类型 |
| 自定义对象 | message | 嵌套消息 |
