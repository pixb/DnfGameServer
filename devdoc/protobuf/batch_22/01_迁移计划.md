## 三、数据结构设计

### 3.1 核心消息结构

#### 3.1.1 流数据

```protobuf
// 流数据
message StreamData {
  int32 id = 1;           // 流数据ID
  bytes data = 2;         // 流数据内容
  int64 timestamp = 3;     // 时间戳
}
```

#### 3.1.2 用户信息

```protobuf
// 用户信息
message UserInfo {
  uint64 guid = 1;        // 用户GUID
  string name = 2;         // 用户名称
  int32 level = 3;         // 用户等级
  int32 job = 4;          // 用户职业
  int64 last_login = 5;    // 最后登录时间
}
```
