### 3.1 核心消息结构

#### 3.1.1 技能信息

```protobuf
// 技能信息
message PtSkill {
  int32 skill_id = 1;     // 技能ID
  int32 level = 2;         // 技能等级
  int32 exp = 3;           // 熟练度
  repeated int32 attributes = 4; // 属性列表
}
```

#### 3.1.2 掉落对象金币信息

```protobuf
// 掉落对象金币信息
message PtDropObjectGold {
  int64 amount = 1;        // 金币数量
  uint64 object_id = 2;    // 掉落对象ID
  float x = 3;             // X坐标
  float y = 4;             // Y坐标
  float z = 5;             // Z坐标
}
```

### 3.2 数据类型映射

| JProtobuf类型 | 标准Protobuf类型 | 备注 |

| float | float | 浮点数类型 |
