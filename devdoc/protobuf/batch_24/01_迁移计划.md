## 三、数据结构设计

### 3.1 核心消息结构

#### 3.1.1 认证信息

```protobuf
// 认证信息
message AuthInfo {
  string username = 1;   // 用户名
  string password = 2;    // 密码
  string token = 3;       // 令牌
  int64 expire_time = 4;  // 过期时间
}
```

#### 3.1.2 角色信息

```protobuf
// 角色信息
message CharacterInfo {
  uint64 guid = 1;       // 角色GUID
  string name = 2;        // 角色名称
  int32 level = 3;        // 角色等级
  int32 job = 4;          // 职业
  int32 gender = 5;       // 性别
  int64 create_time = 6;  // 创建时间
}
```

#### 3.1.3 聊天信息

```protobuf
// 聊天信息
message BaseChat {
  uint64 sender_guid = 1;  // 发送者GUID
  string sender_name = 2;   // 发送者名称
  string content = 3;       // 聊天内容
  int32 type = 4;          // 聊天类型
  int64 timestamp = 5;      // 时间戳
}
```

### 3.2 数据类型映射

| JProtobuf类型 | 标准Protobuf类型 | 备注 |
| :--- | :--- | :--- |
| int | int32 | 整数类型 |
| long | uint64 | 无符号长整数类型 |
| String | string | 字符串类型 |
| boolean | bool | 布尔类型 |
| List<T> | repeated T | 列表类型 |
| 自定义对象 | message | 嵌套消息 |
