# TC005: 发送邮件

## 测试目标
验证玩家能够成功发送邮件，包括向其他玩家发送邮件、发送带附件的邮件等。

## 测试环境
- 服务器状态：正常运行
- 客户端：Java测试客户端
- 网络环境：本地网络

## 测试步骤

### 步骤1: 建立TCP连接
- 客户端与服务器建立TCP连接
- 验证连接成功

### 步骤2: 构造登录请求
- 创建 `REQ_LOGIN` 对象
- 设置 `openid`、`type`、`token`、`platID`、`version`、`clientIP` 等字段

### 步骤3: 编码登录请求
- 使用 `MessageCodec` 编码登录请求
- 验证编码成功

### 步骤4: 发送登录请求
- 向服务器发送编码后的登录请求
- 验证发送成功

### 步骤5: 接收登录响应
- 接收服务器返回的登录响应
- 验证响应数据长度正确

### 步骤6: 解码登录响应
- 使用 `MessageCodec` 解码登录响应
- 验证解码成功，响应类型为 `RES_LOGIN`

### 步骤7: 验证登录成功
- 检查 `RES_LOGIN` 对象中的 `error` 字段是否为 `null`
- 检查 `RES_LOGIN` 对象中的 `authkey` 字段是否不为 `null`
- 验证登录成功

### 步骤8: 构造发送邮件请求
- 创建发送邮件请求对象
- 设置收件人、邮件标题、邮件正文、邮件附件等相关参数

### 步骤9: 编码发送邮件请求
- 使用 `MessageCodec` 编码发送邮件请求
- 验证编码成功

### 步骤10: 发送发送邮件请求
- 向服务器发送编码后的发送邮件请求
- 验证发送成功

### 步骤11: 接收发送邮件响应
- 接收服务器返回的发送邮件响应
- 验证响应数据长度正确

### 步骤12: 验证发送邮件响应
- 使用 `MessageCodec` 解码发送邮件响应
- 验证解码成功，响应类型正确
- 检查响应中的发送结果是否正确
- 验证发送邮件测试通过

## 预期结果
- 登录成功
- 发送邮件请求发送成功
- 收到正确的发送邮件响应
- 邮件发送成功，收件人能够收到邮件
